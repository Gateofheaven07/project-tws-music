# Deezer API Endpoints

## 1. Search Music

https://api.deezer.com/search?q=

Contoh:
https://api.deezer.com/search?q=faded

## 2. Trending / Chart

https://api.deezer.com/chart

## 3. Artist Detail

https://api.deezer.com/artist/{id}

Contoh:
https://api.deezer.com/artist/27

## 4. Artist Top Tracks

https://api.deezer.com/artist/{id}/top

Contoh:
https://api.deezer.com/artist/27/top

## 5. Album Detail

https://api.deezer.com/album/{id}

Contoh:
https://api.deezer.com/album/302127

## 6. Playlist Detail

https://api.deezer.com/playlist/{id}

Contoh:
https://api.deezer.com/playlist/908622995

## 7. Genre List

https://api.deezer.com/genre

## Yang paling wajib untuk MVP:

/search
/chart

## Tambahan untuk fitur advanced:

/artist/{id}
/artist/{id}/top
/album/{id}
/playlist/{id}
/genre
